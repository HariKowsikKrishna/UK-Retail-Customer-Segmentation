# UK Retail Customer Segmentation using K-Means Clustering

## Overview
K-Means clustering analysis on the UCI UK Online Retail dataset 
(805,531 rows) to segment customers by purchasing behaviour.

## Tools Used
- Orange 3 (machine learning and data mining)
- Microsoft Excel (data cleaning and preparation)
- Tableau (visualisation)

## Dataset
UCI Machine Learning Repository — Online Retail II
December 2009 to December 2011, 43 countries

## Methodology
- Variables used: Quantity, Price, Revenue
- K optimised from 2 to 5 clusters (silhouette scoring)
- Optimal K = 3 (silhouette score: 0.327)

## Customer Segments Identified
- Cluster 1: Low value, high volume buyers
- Cluster 2: Medium value buyers  
- Cluster 3: High value, low volume buyers
